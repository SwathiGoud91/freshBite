# FreshBite - Food Ordering Website 🍔

FreshBite is a responsive food ordering website that allows users to explore food items, search for dishes, add items to their cart, and manage their orders through an interactive cart system.

The project is built using **HTML, CSS, and JavaScript** with browser-based Local Storage for maintaining cart data.

## 🚀 Live Demo

🔗 **Website:** https://swathigoud91.github.io/freshBite/

## 📌 Features

* 🍕 Browse available food items
* 🔍 Search food by name, category, and restaurant
* 🛒 Add food items to cart
* ➕ Increase item quantity
* ➖ Decrease item quantity
* ❌ Remove items from cart
* 💾 Persistent cart storage using Local Storage
* 🧾 Automatic order summary calculation
* 🔔 Toast notifications when items are added
* 📱 Responsive user interface

## ⚙️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/SwathiGoud91/freshBite.git
```

2. Navigate to the project folder:

```bash
cd freshBite
```

3. Open `index.html` in your browser.

## 🛒 Application Workflow

1. Users browse food items displayed dynamically from the food dataset.
2. Users search for dishes based on name, category, or restaurant.
3. Clicking **Add To Cart** stores selected items in Local Storage.
4. The cart page retrieves stored items and provides:

   * Food details
   * Quantity management
   * Price calculation
   * Order summary
